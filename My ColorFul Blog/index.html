Introduction to CSS

CSS stands for Cascading Sttyle Sheets.It describes how HTML element are to be displayed on screen, paper, or in other media.
The cascade means that styles can be overridden based on their specificity, allowing for flexible design.

Responsive Web Design

Responsive web design (RWD) ensures websites automatically resize, shrink, or rearrange elements to look good on any device—desktops, tablets, or phones—using one codebase.
